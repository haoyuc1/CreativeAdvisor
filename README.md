# CreativeAdvisor
Connect ChatGPT and Mid-journey

## depandency
```bash
pip install django
pip install requests
pip install pillow
```

## clone
```bash
git clone https://github.com/haoyuc1/CreativeAdvisor.git
cd CreativeAdvisor
```

## run server
```bash
python3 manage.py makemigrations
python3 manage.py migrate
```

```bash
python3 manage.py runserver
```
